\chapter{CONSIDERAÇÕES FINAIS}
\label{cap5}
Utilizar a tecnologia como forma de aumentar o rendimento, diminuir esforço e facilitar o agronegócio é essencial nos dias de hoje. 

O resultado esperado era um painel online para acompanhamento de resultados do MIP na cultura da soja. O painel centraliza toda as informações das análises em um único local, facilitando a visualização da ocorrência de pragas nas plantações. Consequentemente, o painel amenta a eficiência e a eficácia na tomada de decisões. O acesso aos dados pode ser realizado independentemente da localização geográfica do usuário e ainda é possível fazer a instalação da aplicação em dispositivos móveis. 

\section{Limitações}

Pelo fato da API REST não ter sido finalizada, esta aplicação teve que simular uma API localizada em um repositório do Github, na qual segue a mesma estrutura que seria utilizada na API REST final. Essa API fornecia uma JSON com dados fictícios sobre as análises realizadas no Projeto Grãos da EMATER, pois os dados oficiais não poderiam ser divulgados.

Outra limitação, foi a falta de dados fornecidos pela EMATER, como unidades que não participaram do MIP e informações de custo da utilização da técnica do manejo integrado de pragas.  

\section{Trabalhos Futuros}

O painel poderá conter mais informações e análises de outras técnicas, como o Manejo Integrado de Doenças, por exemplo. Além disso, ele poderá se integrar a outras aplicações que podem ser desenvolvidas, como aplicativos de coleta de dados.

Os técnicos da EMATER estão abertos a darem sugestões sobre as possibilidades de análises que a aplicação fornece.
